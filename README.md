✨ Features Built-In

Multiple Modes: Swift navigation panels between Standard mode, Scientific modes, and Live Unit Converters.

Adaptive Screen Sizing: Adapts naturally to tablets, compact smartphone screens, and spacious desktop views.

Keyboard Support: Supports standard physical keypad mappings (Numbers, Backspace, operators, Enter, and escape clearance keys).

Session logs: Stores calculations and formula solutions locally inside history logs.
